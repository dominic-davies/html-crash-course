Inline elements

Don't start on a new line and take only the necessary width.

  • span
  • img
  • a - surrounded text becomes a link
  • strong - surround text, semantic tag for making text stand out.
  • em - surround text, semantic tag for emphasising text (looks italic).
  • form elements - label and inputs - would put in a div (block) to put them on separate lines.

Forms

Forms have an action attribute would submit form to a page like process.php on server.

Basic forms usually have a label followed by input (type - text commonly). Input name helps the server understand with data is linked to which input field.
